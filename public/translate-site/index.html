三步开始使用

简单几步，即可享受 AI 驱动的智能翻译体验

1

安装插件

下载插件包，在 Chrome 中打开扩展管理页，开启开发者模式后加载解压的扩展程序文件夹。

2

配置 API

点击插件图标，选择 AI 模型服务商，填入对应的 API Key（部分免费模型可跳过此步）。

3

开始翻译

打开任意外文网页，点击「翻译当前页面」或选中文本划词翻译，即刻体验 AI 翻译的魅力。
